How to start
------------

1. Make directory for the project.
2. Clone the project.
3. Register and update the submodule.

Example:

	$ mkdir project/ios
	$ cd project/ios
	$ git clone git@github.com:AvalonGameArt/CVS-Tycoon.git cvst
	$ cd cvst
	$ git submodule init
	$ git submodule update --merge
